# История изменений

*[English version](CHANGELOG.md)*

## [0.1.0] - 2026-07-06
### Добавлено
- Первый релиз: модель контракта, валидатор, хранение истории на SQLite
- Обнаружение дрейфа: сравнение недавнего окна с историческим базовым
- Контроль бюджета по стоимости/токенам/латентности
- Автогенерация pytest-регрессионных тестов из сохранённых сбоев
- Команда CI-гейта с настраиваемыми exit-кодами
- Рендер отчётов в text, JSON и JUnit XML
- CLI, объединяющий команды validate / ci-gate / gen-tests / report
- Набор юнит-тестов для валидатора, дрейфа и хранилища
